# 💸 Conversor de Moedas Universal - HTML, CSS & JS

Este projeto é um conversor dinâmico que permite realizar conversões entre múltiplas moedas (Real, Dólar e Euro) de forma bidirecional. O foco principal foi o domínio da manipulação do DOM e a criação de uma lógica de conversão universal.

---

## 💻 Sobre o Projeto

Diferente de conversores simples, este projeto permite que o usuário escolha tanto a moeda de origem quanto a de destino, atualizando interface e valores em tempo real.

### Principais funcionalidades:
- **Conversão Cruzada:** Converta de Dólar para Euro, Euro para Real, e vice-versa.
- **Interface Inteligente:** Troca automática de bandeiras, nomes e símbolos monetários ao selecionar a moeda.
- **Formatação Profissional:** Uso da biblioteca `Intl.NumberFormat` para exibição correta de moedas internacionais.
- **Design Responsivo:** Adaptado para diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica com elementos de formulário e seleção.
- **CSS3:** Estilização moderna e layout flexível.
- **JavaScript (ES6+):** Lógica de conversão baseada em taxas dinâmicas e manipulação avançada do DOM.
