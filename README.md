# 💸 Conversor de Moedas Universal

Este é um projeto de conversor de moedas dinâmico e responsivo, desenvolvido para praticar a manipulação do DOM e lógica de conversão cruzada com JavaScript. O projeto permite converter valores entre **Real, Dólar e Euro** de forma bidirecional.



## 🚀 Funcionalidades

- **Conversão Universal:** Converta de qualquer moeda para qualquer moeda (Ex: Dólar para Euro, Euro para Real, etc).
- **Interface Dinâmica:** As bandeiras, nomes das moedas e símbolos monetários mudam automaticamente ao selecionar a moeda.
- **Formatação de Moeda:** Utiliza a API `Intl.NumberFormat` para exibir os valores corretamente de acordo com a moeda (R$, US$, €).
- **Validação de Entrada:** Aceita apenas números para evitar erros de cálculo.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica do projeto.
* **CSS3:** Estilização com foco em Flexbox para layout responsivo.
* **JavaScript (ES6):** Lógica de conversão, objetos de configuração e manipulação de eventos do DOM.

## 🧠 Desafios Superados

Neste projeto, foquei em aprender:
1.  A diferença entre propriedades como `.value`, `.innerHTML` e `.innerText`.
2.  Como gerenciar múltiplos elementos com classes semelhantes usando `querySelectorAll`.
3.  Lógica de conversão usando uma "moeda base" (Real) para permitir trocas infinitas entre moedas sem encher o código de `if/else`.
